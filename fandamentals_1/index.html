What is JavaScript? - ถูกสร้างขึ้นโดยบริษัท Netscape และเคยถูกเรียกใช้ ECMAscript - เป็นภาษาสคริปต์หลักที่ใช้กับเว็บ ทุกเว็บไซต์ต้องมีจาวาสคริปต์ เป็นภาษาที่สำคัญมาก - ทำงานในฝั่งClient=ผู้ใช้ ทำการInteractive=โต้ตอบ การhover animationต่างๆ ปัจจุบันมีNodejs ทำให้ Js ทำงานได้ทั้งฝั่ง Client และ Server - JavaScript ไม่ใช่ Java What can JavaScript do? - HTML Form Validation - Event Handling - click, hover, etc... - Powers HTML5 Features - Search and form autocomplete - Animations - jQuery, Ajax, and other libraries are all JavaScript Writing javascript in html - file เดียวกัน , ต่าง file กัน ๋JavaScript Variables - Variables should start with letter = เริ่มด้วยตัวอักษร เล็ก, ใหญ่ก็ได้ - They are case sensitive = ตัวเล็ก,ตัวใหญ่ ต่างกัน - var x; = การประกาศตัวแปร x เปล่า ไม่มีค่าอะไรในตัว x - x = 4; = การประการตัวแปร x มีค่า = 4 ไม่ต้องมี var นำหน้าก็ได้ - var x = 4; = แต่เพื่อความมั่นใจ ใส่ var นำหน้าดีกว่า JavaScript Arrays = เก็บค่าได้หลายตัว JavaScript Loops = การรันโค้ดเดิมซ้ำๆ จนกว่าจะเข้าเงื่อนไขที่ถูกต้อง - For loop = จะลูปผ่านบล๊อคตัวเลขทุกตัว แล้วเราจะรู้มันกี่ตัวๆ - While Loop = ลูปแค่ที่เป็นเงื่อนไขเป็นจริง ทำงานก่อนแล้วมารู้ทีหลัง - For in loop = ลูปผ่านอาร์เรย์ ๋JavaScript Functions - ฟังค์ชั่นจะทำงานก็ต้อเมื่อถูกเรียกใช้ ๋JavaScript Objects - ทุกสิ่งทุกอย่างใน JavaScipt จะเป็น Object หมดเลย เช่น String, Date, Array - Object แต่ละตัวก็จะมี Properties สามารถเข้าถึงแบบนี้ objectName.propertyName
